Portshowlio '19 Website

Timeframe: 8 weeks

    Skills:
  • Team lead
  • Lead developer
  • Project manager
    Tools:
  • WordPress
  • GitHub
  • HTML/CSS/JavaScript
  • PHP

The Opportunity

Design and develop the Portshowlio 2019 website featuring graduating student work as well as providing information on the event such as location, date, and time.

Project goals

  • Build a functional and responsive website for the 2019 Portshowlio
  • Translate the site to all screen sizes, implement responsive design
  • Provide documentation on how to fill out the submission form and provide feedback and assistance when necessary

The Process

Divide and conquer:

Acting as team lead, I split our group into two subgroups, design and development. During this time I also acted as project manager by creating Trello and Pinterest boards as well as calendars to keep the team on track.

Design:

After receiving brand guidelines, our designers set out to create the mockups for our website. This process involved multi week iterations for the multiple screens of our site. Design deliverables also included mockups for tablet and mobile view.

Development:

Developers took the design mockups and implemented them using WordPress as our main development platform. I divided the code base, explained the github flow, and each developer began their respective work.

Test and deploy:

Sending out each student a set of login credentials, students uploaded content to the site. Lastly, we deployed the site before the 2019 Portshowlio.

The Solution

As a team, we built the 2019 Portshowlio website. The site is responsive and functional, and references the 2019 Portshowlio brand while also remaining subtle enough to showcase student work. The site will act as a form of future marketing and advertising for the school, as potential students can browse through the site and see what work this program produces. Congratulations grads, we did it!
